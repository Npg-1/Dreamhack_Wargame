간략하게만 그냥 적자면 app.py는 한글을 제외한 영어, 숫자, 특수문자 등의 입력을 받지않고 오직 한글만 출력하는 국뽕 사이트인 것임;;

근데 여기서 보면 `message = unicodedata.normalize("NFKC", message)`를 통해서 사용자의 입력이 좀 무언가 모양이 이상하다면 고쳐서 반환하는데 글자에보면 전각문자, 반각문자 라는 게 있음. 근데 여기서 필터링을 하는 건 반각문자만 필터링을 하기 때문에 전각문자로 페이로드를 짜면 그대로 사용할 수 있음

사용자의 입력을 `<div>%s</div>`을 통해서 출력하기 때문에 여기에 페이로드를 작성하면 됨

반각문자: `{{ self.__init__.__globals__['__builtins__']['open']('/home/hangul/flag').read() }}`
전각문자: `｛｛　ｓｅｌｆ．＿＿ｉｎｉｔ＿＿．＿＿ｇｌｏｂａｌｓ＿＿［＇＿＿ｂｕｉｌｔｉｎｓ＿＿＇］［＇ｏｐｅｎ＇］（＇／ｈｏｍｅ／ｈａｎｇｕｌ／ｆｌａｇ＇）．ｒｅａｄ（）　｝｝`

전각문자는 하나의 문자가 2개의 자리를 차지함

페이로드를 좀 자세히 설명하자면
페이로드: {{ self.__init__.__globals__['__builtins__']['open']('/home/hangul/flag').read() }}

- self: 현재 실행 중인 객체
- .__init__: (__init__)가 붙은 객체를 초기화하는 함수 (생성자)
- .__globals__: 해당 함수가 정의된 전역범위
- \['__builtins__']: 파이썬이 기본으로 제공하는 '도구 상자'에 접근
- ['open']: 도구 상자 안에서 파일을 여는 `open`함수를 꺼냄
- ('/home/hangul/flag').read(): 지정된 경로의 파일을 열어서 출력함
