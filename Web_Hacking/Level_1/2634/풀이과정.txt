About에서 보니까 숏코드 기능을 포스트를 작성할 때 사용하라고 하네요
그리고 숏코드 기능에서 {{include: (URL)}} 처럼 하면 해당 URL의 정보를 불러와서 본문에 표시하는 거 같음

그러면 숏코드 기능을 사용해서 FLAG를 가져오면 되지 않을까 싶은데

https://naver.com -> 네이버는 됨

이게 include 했을 때 보이는 로직이 구체적으로 어떻게 되는지는 모르겠지만 네이버로 했을 때 include가 정상적으로 되긴하는데
결과물에 좀 하자가 있음

{{include: http://host3.dreamhack.games:8540/?p=about}}
해서 사이트 자체를 해보니까 안 되네

그리고 각 포스트에 들어가보니까 사이트명 뒤에 ?p=post&id=3 => 어떤 작업인지(p=post), 몇번 포스트인지(id=3)로 보임

창은 Posts -> p=home
각 포스트에 들어갔을 때 -> p=post&id=(포스트 번호)
포스트를 작성할 때 -> p=write
About에 들어갈 때 -> p=about

숏코드(Shortcode)에서 {{include: ... }} 형식은 주로 특정 위치에 다른 파일, 템플릿, 또는 콘텐츠 뭉치를 불러와서 삽입하는 '참조' 역할을 합니다.

마치 퍼즐 조각을 맞추듯, 한 곳에 정의해 둔 내용을 여러 페이지에서 재사용할 때 아주 유용하죠. 사용하는 시스템(예: Hugo, Jekyll, 혹은 커스텀 CMS 등)에 따라 세부 문법은 다르지만, 핵심 원리는 동일합니다.
